# Rails template by Kuartz

## Features

- [X] Devise
- [X] Force SSL

## Setup

```
rails new \
  --database postgresql \
  -m https://raw.githubusercontent.com/kuartz-org/rails-templates/master/default.rb \
  CHANGE_THIS_TO_YOUR_RAILS_APP_NAME
```
